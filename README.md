# Sepaq-Crawler
Uitlity to crawl the SEPAQ API.

Installation:
```bash
pip install git+https://github.com/AntoinePrv/Sepaq-Crawler
```

Usage:
```bash
python -um sepaq_crawler --arriving 2021-10-10 --leaving 2021-10-15  --distance 100
```

Or for more options:
```bash
python -m sepaq_crawler --help
```
